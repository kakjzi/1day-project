# User Personas

## Persona 1: 고객 (Customer)

| 항목 | 내용 |
|------|------|
| **이름** | 김민수 (대표 고객) |
| **역할** | 매장 방문 고객 |
| **목표** | 빠르고 편리하게 음식 주문 |
| **환경** | 테이블에 설치된 태블릿 |
| **특성** | 다양한 연령대, 기술 친숙도 다양 |

**니즈**:
- 직원 호출 없이 즉시 주문
- 메뉴 사진과 설명으로 선택
- 주문 상태 실시간 확인

**Pain Points**:
- 복잡한 UI는 불편
- 주문 후 상태를 모르면 불안

## Persona 2: 관리자 (Admin)

| 항목 | 내용 |
|------|------|
| **이름** | 박서연 (매장 매니저) |
| **역할** | 매장 운영 관리자 |
| **목표** | 효율적인 주문 관리 및 매장 운영 |
| **환경** | 카운터 PC 또는 태블릿 |
| **특성** | 바쁜 환경에서 빠른 판단 필요 |

**니즈**:
- 신규 주문 즉시 인지
- 테이블별 주문 현황 한눈에 파악
- 메뉴 등록/수정/삭제를 쉽게 처리

**Pain Points**:
- 주문 놓치면 고객 불만
- 여러 테이블 동시 관리 어려움
